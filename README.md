### 🔷 [RecallBase – Vector Memory Engine for LLMs](https://github.com/bluestar404/RecallBase)
> *Python · FastAPI · LangChain · PostgreSQL · FAISS · Docker*

- 🧩 Built a **vector-based memory system** for storing and retrieving contextual data across LLM sessions
- ⚡ Reduced query latency by **up to 40%** using FAISS for efficient similarity search
- 🏗️ Microservice-oriented design: separated embedding, storage, and retrieval layers
- 🐳 Fully containerized with Docker; metadata persistence via PostgreSQL
