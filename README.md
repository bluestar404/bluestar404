> Production-grade AI memory backend with semantic retrieval, background processing, and observability stack.

> An intelligent agentic shopping assistant. Not a chatbot. An agent.

NeuroCart doesn't just search — it reasons. Powered by a **LangGraph DAG** that orchestrates multi-step workflows: intent parsing, product discovery, deal scoring, and structured comparison. The **Gemini API** handles language; the graph handles logic. Multiple modes: smart search, deal optimization, deep comparison.

**`answer-eval`** &nbsp;&nbsp; ![NLP](https://img.shields.io/badge/NLP-Semantic_Scoring-7b2fff?style=flat-square&labelColor=0d0d2b)

> A dual-engine answer evaluation system that measures meaning, not just keywords.

Two scoring modes running in parallel: **BERT embeddings** compute cosine similarity for semantic alignment, while the **Gemini API** applies contextual reasoning for nuanced judgment. Built as an alternative to rubric-based grading — useful anywhere automated assessment needs to understand intent, not just surface matches.
